
# scheduleMe

✈️ UCR FLEAP London Summer 2019 ✈️

A web application made using HTML/CSS, Javascript, and Google Firebase to help friends find the best time to meet.

- Sign-in with Google
- Easy to use UI
- Fast and effective time finder
